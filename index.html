Este es el comienzo del Blog Post, donde detallaremos como ver aspectos intersantes sobre el manejo de Git como interfaz de trabajo remoto

El primer parrafo dentro de este blog post, donde explicaremos algunos conceptos de trabajo remoto de Git Bash, como el manejo de ramas para poder trabajar por separado, para luego devolverlo con merge

El manejo de ramas secudaria sirve para poder sacar cierta parte del codigo master, para poder trabajar independientemente con el proyecto y poder unirlo con una rama master cuando nuestra parte haya sido culminada.
